## 三、pandora-application 关键组件

### 3.1 应用服务示例

```java
@Service
@RequiredArgsConstructor
public class UserApplicationService {
    private final UserRepository userRepository;
    private final DomainEventPublisher eventPublisher;
    private final UserAssembler assembler;

    @Transactional
    public UserDTO createUser(CreateUserCommand cmd) {
        User user = User.create(cmd.getUsername(), cmd.getEmail(), cmd.getPhone());
        user = userRepository.save(user);
        eventPublisher.publishAll(user.getDomainEvents());
        user.clearDomainEvents();
        return assembler.toDTO(user);
    }
}
```

```java
@Configuration
public class MybatisFlexConfig {

    @Bean
    public InsertListener insertListener() {
        return entity -> {
            if (entity instanceof BaseEntity base) {
                base.setCreateTime(LocalDateTime.now());
                base.setUpdateTime(LocalDateTime.now());
                LoginUser user = LoginUserHolder.get();
                if (user != null) {
                    base.setCreateBy(user.getUserId());
                    base.setUpdateBy(user.getUserId());
                }
            }
        };
    }

    @Bean
    public UpdateListener updateListener() {
        return entity -> {
            if (entity instanceof BaseEntity base) {
                base.setUpdateTime(LocalDateTime.now());
                LoginUser user = LoginUserHolder.get();
                if (user != null) base.setUpdateBy(user.getUserId());
            }
        };
    }
}
```

### 4.5 分布式锁

```java
@Service
@RequiredArgsConstructor
public class DistributedLockService {
    private final RedissonClient redissonClient;

    public <T> T executeWithLock(String key, long waitSec, long leaseSec, Supplier<T> action) {
        RLock lock = redissonClient.getLock("lock:" + key);
        try {
            if (!lock.tryLock(waitSec, leaseSec, TimeUnit.SECONDS)) {
                throw new BusinessException(ResultCode.LOCK_FAILED);
            }
            return action.get();
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new BusinessException(ResultCode.LOCK_INTERRUPTED);
        } finally {
            if (lock.isHeldByCurrentThread()) lock.unlock();
        }
    }
}
```

### 4.7 JWT 工具（JJWT 0.12.6）

```java
public class JwtUtils {
    public String generate(Map<String, Object> claims, long ttlSec, SecretKey key) {
        return Jwts.builder()
                   .claims(claims)
                   .expiration(Date.from(Instant.now().plusSeconds(ttlSec)))
                   .signWith(key, Jwts.SIG.HS256)
                   .compact();
    }

    public Claims parse(String token, SecretKey key) {
        return Jwts.parser().verifyWith(key).build().parseSignedClaims(token).getPayload();
    }
}
```

### 6.2 Token 生成

```java
public LoginResponse generateBToken(AdminUser user) {
    String jti = UUID.randomUUID().toString();
    long tgen = tgenService.current(user.getId(), UserType.B_USER);

    Map<String, Object> claims = Map.of(
        "jti", jti,
        "sub", user.getId().toString(),
        "userType", UserType.B_USER.getCode(),
        "roles", user.getRoleIds(),
        "deptId", user.getDeptId(),
        "dataScope", user.getDataScope().getCode(),
        "tgen", tgen
    );
    String accessToken = jwtUtils.generate(claims, accessTtl, bKey);

    String refreshToken = UUID.randomUUID().toString();
    tokenWhitelistService.save(refreshToken, user.getId(), refreshTtl);
    return new LoginResponse(accessToken, refreshToken, accessTtl);
}
```

### 6.3 黑名单 / 代次

- 登出：`blacklist:{jti}` → Redis，TTL 等于 AccessToken 剩余时间
- 强制踢人：`INCR tgen:{userType}:{userId}`，网关或资源服务器校验 claim `tgen` < 当前值则拒绝

### 7.2 全局鉴权过滤器

```java
@Component
public class AuthGlobalFilter implements GlobalFilter, Ordered {

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String path = exchange.getRequest().getURI().getPath();
        if (whiteList.match(path)) return chain.filter(exchange);

        String token = exchange.getRequest().getHeaders().getFirst(HttpHeaders.AUTHORIZATION);
        if (token == null || !token.startsWith("Bearer ")) return unauthorized(exchange);

        Claims claims = jwtUtils.parse(token.substring(7), key);
        return tokenBlacklistService.exists(claims.get("jti", String.class))
            .flatMap(blocked -> blocked ? unauthorized(exchange) : chain.filter(exchange));
    }

    @Override public int getOrder() { return -100; }
}
```

## 十、开发扩展指引

| 任务 | 涉及模块 | 建议步骤 |
|------|---------|---------|
| 新增一个聚合根 | domain → application → infrastructure → api | 1) domain 写聚合/值对象/仓储接口；2) application 写应用服务与 DTO；3) infrastructure 写 MyBatis-Flex Mapper + Repository 实现；4) api 暴露 Controller |
| 新增一个微服务 | pandora-service | 新建 `pandora-service-xxx` 模块，依赖 `pandora-api + pandora-infrastructure + pandora-common-cloud` |
| 新增一种登录方式 | pandora-auth | 新建 `XxxAuthProvider implements AuthProvider` 并注册为 Bean |
| 新增一种缓存策略 | pandora-infrastructure | 通过 `RedissonClient` 提供的 RMap / RBucket / RBloomFilter / RRateLimiter |
| 新增文件存储后端 | pandora-application + pandora-infrastructure | 应用层保留 `FileService` 抽象，基础设施层实现新的 Bean 并切换 |
