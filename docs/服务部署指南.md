# 服务部署指南

> 本文档覆盖单体服务和微服务的生产环境部署方案。
> 基线：JDK 25 + Spring Boot 4.0.5 + Spring Cloud Alibaba 2025.1.0.0 + MyBatis-Flex 1.11.6 + Redisson 4.3.1

---

## 一、构建

### 1.1 打包

```bash
# 全量打包
mvn clean package -DskipTests

# 产物：
# pandora-auth/target/pandora-auth-1.0.0.jar                      # Auth 服务
# pandora-start/target/pandora-start-1.0.0.jar                    # 单体服务
# pandora-gateway/target/pandora-gateway-1.0.0.jar                # 网关
# pandora-service/pandora-service-user/target/pandora-service-user-1.0.0.jar
```

按环境打包：

```bash
mvn clean package -P dev  -DskipTests
mvn clean package -P test -DskipTests
mvn clean package -P prod -DskipTests
```

### 1.2 Docker 镜像（通用 Dockerfile）

```dockerfile
FROM eclipse-temurin:25-jre-alpine
WORKDIR /app
COPY target/*.jar app.jar
ENV JAVA_OPTS="-XX:+UseZGC -XX:+ZGenerational -Dspring.threads.virtual.enabled=true -XX:MaxDirectMemorySize=256m"
EXPOSE 8080
ENTRYPOINT ["sh","-c","exec java $JAVA_OPTS -jar app.jar"]
```

构建：

```bash
docker build -t pandora-auth:1.0.0     -f Dockerfile ./pandora-auth
docker build -t pandora-start:1.0.0    -f Dockerfile ./pandora-start
docker build -t pandora-gateway:1.0.0  -f Dockerfile ./pandora-gateway
docker build -t pandora-service-user:1.0.0 -f Dockerfile ./pandora-service/pandora-service-user
```

## 二、单体部署

### 2.1 架构

```
           ┌──────────────┐      ┌────────────────────────┐
用户  ───→ │ Nginx (443)  │────→ │ pandora-start :8080    │────→ MySQL 9 + Redis + MinIO
           └──────────────┘      └────────────────────────┘
                                         │
                                 ┌───────┴──────────────┐
                                 │ pandora-auth :9100   │
                                 │（Spring AS 7.0.4）    │
                                 └──────────────────────┘
```

### 2.2 中间件安装

#### MySQL 9.x

```bash
docker run -d --name mysql \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=123456 \
  -v /data/mysql:/var/lib/mysql \
  mysql:9.0 --character-set-server=utf8mb4 --collation-server=utf8mb4_general_ci

docker exec -i mysql mysql -u root -p123456 < docs/sql/schema.sql
docker exec -i mysql mysql -u root -p123456 ddd_platform < docs/sql/schema-auth.sql
docker exec -i mysql mysql -u root -p123456 ddd_platform < docs/sql/schema-rbac.sql
docker exec -i mysql mysql -u root -p123456 ddd_platform < docs/sql/schema-member.sql
```

#### Redis 7

```bash
docker run -d --name redis \
  -p 6379:6379 \
  -v /data/redis:/data \
  redis:7-alpine --appendonly yes --requirepass "your-redis-password"
```

#### MinIO

```bash
docker run -d --name minio \
  -p 9000:9000 -p 9001:9001 \
  -v /data/minio:/data \
  -e "MINIO_ROOT_USER=minioadmin" \
  -e "MINIO_ROOT_PASSWORD=minioadmin" \
  minio/minio server /data --console-address ":9001"
```

控制台 http://your-server:9001 创建 `corp-web` 桶。

### 2.3 部署服务

```bash
mkdir -p /opt/pandora/{auth,start,logs}

cp pandora-auth/target/pandora-auth-1.0.0.jar       /opt/pandora/auth/app.jar
cp pandora-start/target/pandora-start-1.0.0.jar     /opt/pandora/start/app.jar
```

#### 启动 Auth 服务

```bash
JAVA_OPTS="-Xms512m -Xmx1g -XX:+UseZGC -XX:+ZGenerational -Dspring.threads.virtual.enabled=true"

nohup java $JAVA_OPTS -jar /opt/pandora/auth/app.jar \
  --spring.profiles.active=prod \
  --spring.datasource.url="jdbc:mysql://localhost:3306/ddd_platform?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true" \
  --spring.datasource.username=root \
  --spring.datasource.password=your-db-password \
  --spring.data.redis.host=localhost \
  --spring.data.redis.database=1 \
  --spring.data.redis.password=your-redis-password \
  --auth.jwt.secret=your-production-secret-key-at-least-256-bits \
  > /opt/pandora/logs/auth.log 2>&1 &

echo $! > /opt/pandora/auth/app.pid
```

#### 启动 Start（单体）服务

```bash
nohup java $JAVA_OPTS -jar /opt/pandora/start/app.jar \
  --spring.profiles.active=prod \
  --spring.datasource.dynamic.datasource.master.url="jdbc:mysql://localhost:3306/ddd_platform?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true" \
  --spring.datasource.dynamic.datasource.master.username=root \
  --spring.datasource.dynamic.datasource.master.password=your-db-password \
  --spring.data.redis.host=localhost \
  --spring.data.redis.database=1 \
  --spring.data.redis.password=your-redis-password \
  --auth.jwt.secret=your-production-secret-key-at-least-256-bits \
  > /opt/pandora/logs/start.log 2>&1 &

echo $! > /opt/pandora/start/app.pid
```

### 2.4 Nginx 反向代理

```nginx
upstream pandora_start { server 127.0.0.1:8080; }
upstream pandora_auth  { server 127.0.0.1:9100; }

server {
    listen 443 ssl http2;
    server_name your-domain.com;

    ssl_certificate     /etc/nginx/ssl/fullchain.pem;
    ssl_certificate_key /etc/nginx/ssl/privkey.pem;

    location /api/ {
        proxy_pass http://pandora_start;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /auth/ {
        proxy_pass http://pandora_auth;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location ~ ^/(swagger-ui|v3/api-docs)/ {
        proxy_pass http://pandora_start;
    }

    location / {
        root /opt/pandora/web;
        try_files $uri $uri/ /index.html;
    }
}
```

### 2.5 Systemd 服务（可选）

```ini
# /etc/systemd/system/pandora-auth.service
[Unit]
Description=Pandora Auth Server
After=network.target mysql.service redis.service

[Service]
Type=simple
User=pandora
WorkingDirectory=/opt/pandora/auth
ExecStart=/usr/bin/java -Xms512m -Xmx1g -XX:+UseZGC -XX:+ZGenerational -jar app.jar --spring.profiles.active=prod
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
systemctl enable pandora-auth
systemctl start pandora-auth
systemctl status pandora-auth
```

## 三、微服务部署

### 3.1 架构

```
                          ┌───────────────────────┐
                    ┌────→│ pandora-auth :9100    │
                    │     │ Spring AS 7.0.4       │
                    │     └───────────────────────┘
        ┌───────────┴──┐  ┌────────────────────┐
用户 ──→│ Gateway :8888│─→│ service-user :8081 │──→ MySQL 9
        │ SCG WebFlux  │  └────────────────────┘    Redis（Redisson）
        └──────┬───────┘  ┌────────────────────┐    MinIO
               │          │ 其他业务微服务      │    Kafka（可选）
               │          └────────────────────┘    MongoDB（可选）
               │
      ┌────────┴──────────┐
      │ Nacos 集群 :8140/8550/8960 │
      │ Sentinel Dashboard :8858  │
      └───────────────────────────┘
```

### 3.2 Nacos 部署（生产推荐集群）

```bash
# 单机（开发 / 测试）
docker run -d --name nacos \
  -p 8848:8848 -p 9848:9848 \
  -e MODE=standalone \
  -e SPRING_DATASOURCE_PLATFORM=mysql \
  -e MYSQL_SERVICE_HOST=mysql \
  -e MYSQL_SERVICE_DB_NAME=nacos_config \
  -e MYSQL_SERVICE_USER=root \
  -e MYSQL_SERVICE_PASSWORD=123456 \
  nacos/nacos-server:v2.4.0
```

生产环境建议 **3 节点集群**，Data ID / 客户端地址使用：`192.168.50.168:8140,8550,8960`（或自定义）。

### 3.3 Nacos 配置（Group = 环境 Profile）

登录 Nacos（http://<host>/nacos），按环境创建配置：

**Group：`prod`（严禁 `DEFAULT_GROUP`）**

**Data ID：`pandora-auth-server-prod.yml`**

```yaml
spring:
  datasource:
    url: jdbc:mysql://db-host:3306/ddd_platform?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: your-db-password
  data:
    redis:
      host: redis-host
      port: 6379
      database: 1
      password: your-redis-password

auth:
  jwt:
    secret: your-production-jwt-secret-key-at-least-256-bits-long
    expiration: 7200
    refresh-expiration: 604800
  sms:
    mock-enabled: false
```

**Data ID：`pandora-service-user-prod.yml`**

```yaml
spring:
  datasource:
    dynamic:
      primary: master
      datasource:
        master:
          url: jdbc:mysql://db-host:3306/ddd_platform?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
          username: root
          password: your-db-password
          driver-class-name: com.mysql.cj.jdbc.Driver
        slave:
          url: jdbc:mysql://db-slave-host:3306/ddd_platform?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
          username: readonly
          password: your-slave-password
          driver-class-name: com.mysql.cj.jdbc.Driver
  data:
    redis:
      host: redis-host
      port: 6379
      database: 1
      password: your-redis-password

auth:
  jwt:
    secret: your-production-jwt-secret-key-at-least-256-bits-long
```

**Data ID：`pandora-gateway-prod.yml`**（注意使用新属性前缀）

```yaml
spring:
  cloud:
    gateway:
      server:
        webflux:
          discovery:
            locator:
              enabled: true
              lower-case-service-id: true
          routes:
            - id: pandora-auth
              uri: lb://pandora-auth-server
              predicates:
                - Path=/auth/**
            - id: pandora-service-user
              uri: lb://pandora-service-user
              predicates:
                - Path=/api/users/**
          default-filters:
            - DedupeResponseHeader=Access-Control-Allow-Credentials Access-Control-Allow-Origin
```

### 3.4 启动顺序

```bash
JAVA_OPTS="-Xms512m -Xmx1g -XX:+UseZGC -XX:+ZGenerational -Dspring.threads.virtual.enabled=true"
NACOS="192.168.50.168:8140,192.168.50.168:8550,192.168.50.168:8960"

# 1. Auth（注册 Nacos）
java $JAVA_OPTS -jar pandora-auth-1.0.0.jar \
  --spring.profiles.active=prod \
  --NACOS_ENABLED=true \
  --NACOS_SERVER_ADDR="$NACOS"

# 2. User 微服务
java $JAVA_OPTS -jar pandora-service-user-1.0.0.jar \
  --spring.profiles.active=prod \
  --NACOS_SERVER_ADDR="$NACOS"

# 3. Gateway（最后启动）
java $JAVA_OPTS -jar pandora-gateway-1.0.0.jar \
  --spring.profiles.active=prod \
  --NACOS_SERVER_ADDR="$NACOS"
```

## 四、生产环境安全清单

| 项目 | 说明 |
|------|-----|
| 修改 JWT Secret | 至少 256 位随机字符串，环境变量注入 |
| 修改数据库密码 | 不使用 123456 |
| 修改 Redis 密码 | `requirepass` 必开 |
| 修改 MinIO 密码 | 不使用 minioadmin |
| 关闭 SMS Mock | `auth.sms.mock-enabled=false`，接入真实短信 |
| 关闭 Swagger | `springdoc.api-docs.enabled=false` |
| 配置 HTTPS | Nginx 配置 SSL 证书 |
| 配置社交登录 | 真实 微信 / QQ / 企业微信 凭据 |
| JVM 参数 | `-Xms512m -Xmx1g -XX:+UseZGC -XX:+ZGenerational`（JDK 25） |
| 日志 | logback-spring.xml 文件滚动 + 级别 |
| Nacos Group | 必须按 `dev/test/prod` 隔离，严禁 DEFAULT_GROUP |
| 网关属性前缀 | 必须 `spring.cloud.gateway.server.webflux.*` |

## 六、运维命令

```bash
# 日志
tail -f /opt/pandora/logs/auth.log
tail -f /opt/pandora/logs/start.log

# 重启
kill $(cat /opt/pandora/auth/app.pid)
# 重新执行启动命令

# 端口占用
ss -tlnp | grep -E '9100|8080|8888|8081'

# JVM GC（JDK 25）
jstat -gc $(cat /opt/pandora/auth/app.pid) 5000
# 查看 ZGC 日志
java -Xlog:gc*:file=gc.log:time,uptime,level,tags -XX:+UseZGC -XX:+ZGenerational -jar app.jar
```
